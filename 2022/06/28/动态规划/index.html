动态规划

1. 概要

动态规划中每一个状态一定是由上一个状态推导出来的

解题步骤：

  1. 确定 dp 数组（dp table）以及下标的含义
  2. 确定递推公式
  3. dp 数组如何初始化
  4. 确定遍历顺序
  5. 举例推导 dp 数组

2. 题目训练

2.1 斐波那契数

🔗题目链接

2.1.1 思路

  1. 确定 dp 数组（dp table）以及下标的含义

    dp[i] ➡️ 第 i 次斐波那契数的值

  2. 确定递推公式

    dp[i] = dp[i-1] + dp[i-2]

  3. dp 数组如何初始化

    • d[0] = 0
    • d[1] = 1

  4. 确定遍历顺序

    dp[i] = dp[i-1] + dp[i-2]

    要先确定前两位的数值，才能确定第三位的数值，所以是从前往后便利

  5. 举例推导 dp 数组

    0，1，1，2，3，5，8，13，21…
