动态规划

1. 概要

动态规划中每一个状态一定是由上一个状态推导出来的

解题步骤：

  1. 确定 dp 数组（dp table）以及下标的含义
  2. 确定递推公式
  3. dp 数组如何初始化
  4. 确定遍历顺序
  5. 举例推导 dp 数组

2. 题目训练

2.1 斐波那契数

🔗题目链接

2.1.1 思路

  1. 确定 dp 数组（dp table）以及下标的含义

    dp[i] ➡️ 第 i 次斐波那契数的值

  2. 确定递推公式

    dp[i] = dp[i-1] + dp[i-2]

  3. dp 数组如何初始化

    • d[0] = 0
    • d[1] = 1

  4. 确定遍历顺序

    dp[i] = dp[i-1] + dp[i-2]

    要先确定前两位的数值，才能确定第三位的数值，所以是从前往后便利

  5. 举例推导 dp 数组

    0，1，1，2，3，5，8，13，21…

2.2 爬楼梯

🔗题目链接

2.2.1 思路

  1. 确定 dp 数组（dp table）以及下标的含义

    dp[i] ➡️ 爬到底 i 阶有多少种方法

  2. 确定递推公式

    dp[i-1] ➡️ 再爬一阶 ➡️ dp[i]

    dp[i-2] ➡️ 再爬两阶 ➡️ dp[i]

    综上 dp[i] = dp[i-1] + dp[i-2]

  3. dp 数组如何初始化

    dp[1] = 1

    dp[2] = 2

  4. 确定遍历顺序

    dp[i] = dp[i-1] + dp[i-2]

    所以从前向后遍历

  5. 举例推导 dp 数组

    0, 1, 2, 3, 5, 8..

2.3 使用最小花费爬楼梯

🔗题目链接

2.3.1 思路

  1. 确定 dp 数组（dp table）以及下标的含义

    dp[i] ➡️ 爬到第 i 阶楼梯的花费

  2. 确定递推公式

    ❌：每次可以选择爬一格 or 爬两格，所以要取决于 dp[i-1], dp[i-2] 中较小的那个

    • dp[i] = dp[i-1] + nums[i]
    • dp[i] = dp[i-1] + nums[i+1]

    dp[i] = dp[i-1] + dp[i-2] + cost[i]

  3. dp 数组如何初始化

    在开始时，你可以选择从下标为 0 或 1 的元素作为初始阶梯。

    • dp[0] = cost[0]
    • dp[i] = cost[1]

  4. 确定遍历顺序

    前 ➡️ 后

  5. 举例推导 dp 数组

    cost = [1, 100, 1, 1, 1, 100, 1, 1, 100, 1]

    dp = [1,100,2,3,3,103,4,5,104,6]
