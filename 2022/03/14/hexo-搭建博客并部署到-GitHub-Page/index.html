hexo 搭建博客并部署到 GitHub Page

Mar 14, 2022 • ritsurin • View

1. 确保电脑有 node 环境

输出如下：

没有 node 环境就百度装下，很快

2. 安装 hexo

中文官网地址：https://hexo.io/zh-cn/

3. 在你喜欢的位置新建一个文件夹

该文件夹用于存储 hexo 文件，以后本地上的博客内容也在里面

==命令行进入该文件夹==

需要等待亿小会

4. 安装部署插件

5. 测试本机 ssh key

若输出 Hi ...，则说明本机 ssh 配置完成

若输出其他，表示未配置 ssh

如何配置可参考该文章

6. 在 GitHub 上创建仓库

仓库名为：你的用户名.github.io

比如我的用户名为 liskarmw，则仓库名为 liskarmw.github.io

==不可更改==

7. 打开 hexo 本地文件夹，修改 _config.yml 配置

8. 命令行输入命令测试

依次输入

等待 push 完成，在浏览器地址栏输入刚刚的仓库名，测试是否能正常访问

9. 开始写作

文章路径为 source/_post

推送

10. 自用主题

https://butterfly.js.org/
